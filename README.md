# Bunker - implementation of card game "Bunker"

# WARNING

This code may be buggy.

## Installation

Download this repo.

Use the offical cards for special and additional information.

## Instructions

cd to project repo, run :

```shell
python3 main.py
```

Enter number of players -> enter players folder.
All files in this directory are player cards.
Send player cards to your friends.
Get some fun?
